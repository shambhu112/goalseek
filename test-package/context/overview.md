## Goal

Predict the **irrigation need** of a crop field (multiclass classification) given a set of soil, weather, crop, and farm-management features.

---

## Evaluation

Submissions are scored on **Balanced Accuracy** — the fraction of correctly predicted labels.

Top leaderboard scores are in the ~0.982 range, indicating high separability in the data.

**Submission format:**

